## Configure ENV variables
Need the following env variables in a .env file at root
```
OPENAI_API_KEY=
FREEPLAY_KEY=
FREEPLAY_PROJECT_ID="45e20940-322a-4cb8-810f-3934a56aa5e2"
FREEPLAY_URL="https://dev.freeplay.ai/api"
```

## Run the client
```
/bin/bash runApp.sh
```
